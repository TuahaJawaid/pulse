# Pulse

Pulse is a public AI news signal dashboard. It aggregates AI stories from Hacker News, Reddit, RSS publishers, and ArXiv, then lets readers search, filter, sort, save, and share the feed.

## Features

- Search across titles, summaries, sources, companies, and categories
- Filter by category, source, and trending company
- Sort by latest, signal score, discussion count, or source score
- Save stories locally in the browser
- Share URLs with active search and filter state
- Rule-based signal scores with no paid AI calls
- About page with source and refresh details

## Development

Use Node.js 20.9 or newer.

```bash
npm install
npm run dev
```

Open http://localhost:3000.
